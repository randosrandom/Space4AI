\doxysubsection{Detailed Description}
Abstract class to represent the parent of all performance models that will inherit from it. 

Abstract class used to represent a performance model for predicting the response time of a \mbox{\hyperlink{classSpace4AI_1_1Partition}{Partition}} object deployed onto a given \mbox{\hyperlink{classSpace4AI_1_1Resource}{Resource}}.
