\doxysubsection{Detailed Description}
Defines the classes for the performance evaluators. 

Polymorphism is used. Note that some Performance Models constructor could accept parameters that will not be used by the class. This trick is required to provide the same interface to the object factory builder. a-\/\+ML library predictor are used as pybind11 classes, implemented with the Meyer\textquotesingle{}s singleton pattern in the file \mbox{\hyperlink{PerformancePredictors_8hpp}{Performance\+Predictors.\+hpp}}
